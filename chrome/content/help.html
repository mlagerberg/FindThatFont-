FindThatFont!

This is the help file for FindThatFont!

About FindThatFont!

FindThatFont! is an open source tool for viewing and organizing your fonts. You can use it to see how a certain text looks in a certain font, to compare different fonts and to categorize all fonts on your system.

You can fork FindThatFont! on Github.

How to use

  • List fonts
    Upon startup, FindThatFont! displays a list of all fonts, grouped in categories like 'Serif', 'Sans serif', 'Handwritten', 'Comic', etc. Using the dropdown boxes above the list you may select a subset of fonts to be displayed. Note that fonts may appear in more than one category at the same time. Select type 'Uncategorized' to see a list of fonts that are not yet classified. Note that you can create your own categories by selecting the menu item 'Fonts > Edit classes...'.
  • Categorize fonts
    If you have fonts that are not categorized, or fonts that are mistakenly miscategorized, you can correct by double clicking on the font. This will open the edit font list' dialog, which you can also open by selecting the menu 'Fonts' > 'Edit font list...' or by pressing 'F3'. In this dialog you can select all classes to which you want to add the font, and you have the option to exclude the font from displaying in the font list and exported HTML documents. This option will not uninstall the font from you system!
  • Exporting and printing
    Using the option in menu 'File' > 'Export as HTML...' (or 'Ctrl+S'), you can save the generated fontlist to disk. This will generate a W3C-compliant web page listing the selected fonts with a preview of each of the fonts. You can directly print the font list by selecting 'Print...' in the 'File'.
  • Appearance
    To change the colors of the fonts or to change the sentence used in the examples, move to 'Fonts > Appearance...', or press 'F9'. In the appearing window you can change the colors or enter a different preview sentence. The default sentence is "My Faxed Joke Won A Pager In A Cable TV Quiz Show", a sentence that contains every letter in the alphabet.

Compatibility:

This program is build to run on Windows and Linux.
Note that, due to the nature of XUL, FindThatFont! can only display unicode fonts, which means that some fonts, like WingDings, are displayed with a default font, often Times New Roman.

Version history:

  • 1.0 — Many changes to the program. Best improvements: speed, threading and editable categories. The program is a lot faster when viewing a single category, due to the new way in which the fontlist is stored. Second: threading is implemented so the interface no longer locks up when refreshing the font list, and you can interrupt refreshing the font list by pressing the Escape key or the cancel button on the statusbar. Third improvement: categories are now editable by the user. The user can choose which categories to show or hide, and can change the names and descriptions and thus is able to create new categories.
    The interface is also improved: with toolbar buttons for 'All classes', 'favorites' and 'uncategorized items'; in the preference window the user can toggle wether or not to show labels on the toolbar buttons; a description of a class is showed in a tooltip on the classname in the list view; searching is improved to function similar to the Firefox webbrowser.
    Numerous minor bugs are solved.
  • 0.3 — Second official release. New features: option to add fonts to a shortlist of favorites, to enable the user to compare certain fonts more easily, and a search bar, with which users can find fonts by (partial) name. Reorganized menus. Solved a bug that caused fonts to appear in default system font when displayed in certain font size.
  • 0.2 — First official release. Includes extension manager (and thus allows to install extensions). Class headers of empty classes are shown in gray. Context menu added to font list. New font classes defined that should better fit your needs. Tooltip appears on the font list that shows in which classes the font under the cursor is put. Stores fontlist.ini in profile, so that it is not overwritten when installing a newer version of FindThatFont!. Allows to print the font list directly from the program rather than exporting to HTML file first. Includes 'check for updates' option.
  • 0.1 Beta — First Beta release. Exports to HTML. Supports custom colors and custom preview sentence.
